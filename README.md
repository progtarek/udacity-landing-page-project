# Udacity Landing Page Project

Simple multi-section landing page, with a dynamically updating navigational menu based on the amount of content that is added to the page, written in html, css and JS.

### Project structure

- css
  - styles.css
- js
  - app.js
- index.html
- README.md

## Running

To launch this website, simply clone the repo and open the `index.html` file in the browser.
